# 🏛️ Sthenos

**Sthenos** (Greek: *Σθένος* — strength, might) is a modern, high-performance personal fitness tracking application. Inspired by the fluid, community-driven experience of Strava, Sthenos is built to eliminate friction in the gym and focus on what truly matters: **your progress.**

---

## 🎯 Project Vision
Most fitness apps are either overly cluttered or dependent on a perfect internet connection. **Sthenos** solves this by implementing a *Local-first* architecture—log your sets instantly with zero latency, while cloud synchronization happens seamlessly in the background.

## ✨ Key Features

* **⚡ Rapid Logging:** Optimized UI for recording sets, reps, and weight in record time (less typing, more lifting).
* **📊 Progressive Overload Analytics:** Automatic tracking of training volume and intensity with visual progress charts.
* **📡 24/7 Cloud Availability:** Powered by Supabase to ensure your data is secure and accessible across all your devices.
* **📶 Offline-First Experience:** Works flawlessly in basement gyms or areas with poor reception. Data is cached locally and synced once you're back online.
* **🏆 PR Vault:** Automatically detects and highlights your All-Time Personal Records.
* **🌑 Elite Dark Mode:** A sleek, high-contrast aesthetic designed for focus and battery efficiency.

## 🛠️ Tech Stack

| Technology | Purpose |
| :--- | :--- |
| **Expo (React Native)** | Cross-platform mobile development |
| **Supabase** | PostgreSQL Database & Authentication |
| **TanStack Query** | Data fetching, caching, and synchronization |
| **NativeWind** | Tailwind CSS styling for Native UI |
| **SQLite / Legend-State** | Local persistence for offline reliability |
